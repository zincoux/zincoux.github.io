起源

2019-05-30

年前辞了职，到现在已经两个多月。期间经历了搬家、迁户，被搞得焦头烂额。目前事已基本妥当，算是安顿了下来。住处附近有河，很像小时候家附近的那条。

年后主要在学 Python 和 Swift ，为之后的自由职业之路增加些想象空间。离职前的那段时期，脑中经常出现各种新奇想法，现在是时候让他们一个个实现了。自学的过程困难重重，但都属于那种克服之后会感到提升的那种，这种兴奋感会让人想要追求“跃进”、追求“多快好省”，不知是好是坏。

技能完善之后，重心会放在生成有价值的产出上，包括应用，服务，文章……

未来模模糊糊，是他该有的样子。

是为起源。
